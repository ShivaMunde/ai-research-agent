# AI Research Agent

This project implements a Python-based AI research agent that automatically searches the web and summarizes results using an LLM.

## Features

- Web search using DDGS
- AI summarization using Groq LLM API
- Automated research pipeline

## Workflow

User Query → Web Search → AI Model → Summary

## Tech Stack

Python  
Groq API  
DDGS Search  

## Example

Input:
AI agents in automation

Output:
• AI agents automate complex workflows  
• LLM-powered agents assist with research tasks  
• Multi-agent systems improve decision making
